# Robert Caro Method

[English](./README.md) | **简体中文**

本 AgentSkill 提炼自传奇传记作家 Robert A. Caro 的研究方法。通过调用该技能，可将 OpenClaw AI 转化为结构化权力分析工具，适用于评估复杂的权力动态、组织机制与领导者战略决策。

> ⚠️ 注：本文档与本技能的中文输出均遵循 `tech-doc-style-chinese` 写作规范，保持克制、准确，避免使用互联网黑话或情绪化表达。

## 核心分析框架

本技能的底层逻辑建立在对 5 部巨著（合计 4,697 页、超 200 万词）的系统性研究范式之上：

- **翻阅每一页（Turn Every Page）**：基于极高的颗粒度进行穷尽式分析，拒绝表面归因。
- **权力的获取与行使**：重点拆解权力杠杆，解析主体如何利用规则漏洞或资源整合来巩固地位。
- **制度的物理学**：将机构（如参议院、公共职能部门、企业委员会）视为物理机制，分析其中的利益交换与规则博弈。
- **对无权者的影响**：在评估宏观决策时，永远叠加一层微观视角，衡量权力运作对普通个体造成的真实代价。

## 安装说明

在 OpenClaw 中安装此技能有两种方式：

### 方式一：克隆仓库（推荐）

直接将此仓库克隆到 OpenClaw 的 skills 目录下：

```bash
cd ~/.openclaw/workspace/skills/
git clone https://github.com/kent666/openclaw-skill-robert-caro-method.git robert-caro-method
方式二：使用安装包





在 Releases 中下载 robert-caro-method.skill 文件（或直接下载源码压缩包）。



将文件移动至 ~/.openclaw/workspace/skills/ 目录。

触发与适用场景

安装该技能后，在对话中提及以下关键词即可触发专用分析模式：





「Robert Caro」 或 「卡罗方法」



「权力分析」 或 「制度机制分析」

适用场景示例：





剖析复杂的公司政治或组织架构重组。



评估长线商业战略的潜在后果与利益博弈。



深度研究特定历史人物或机构的发展轨迹。
