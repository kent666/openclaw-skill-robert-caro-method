Robert Caro Method

English | 简体中文

An OpenClaw AgentSkill based on the exhaustive research methodologies of legendary biographer Robert A. Caro. This skill transforms the AI into a structural power analyst capable of evaluating power dynamics, institutional mechanics, and the strategic decisions of leaders.

Core Methodology

The skill draws upon the analytical rigor required to produce Caro's 5 monumental books (totaling 4,697 pages and 2,047,031 words):





"Turn Every Page": Exhaustive attention to detail, rejecting surface-level explanations.



The Mechanics of Power: How power is acquired, maintained, and wielded through institutional leverage.



Institutional Physics: Treating organizations (like the US Senate or urban planning authorities) as mechanical systems with distinct rules and vulnerabilities.



The Human Cost: Always measuring the impact of high-level decisions on the powerless.

Installation

You can install this skill in OpenClaw using one of the following methods:

Method 1: Clone the repository (Recommended)

Run this in your terminal to clone the skill directly into your OpenClaw workspace:

cd ~/.openclaw/workspace/skills/
git clone https://github.com/kent666/openclaw-skill-robert-caro-method.git robert-caro-method

Method 2: Manual Installation





Download the repository files.



Move the entire folder (or the .skill package) into your OpenClaw skills directory: ~/.openclaw/workspace/skills/
